# hateblog-backend

はてなブックマークのエントリーを新着順・人気順で閲覧できる「はてブログ」のバックエンドAPI。

**技術スタック**: Go 1.25+ / PostgreSQL 18 / Redis 8.4 / Docker

## クイックスタート

```bash
# VSCode で Dev Container を開く（推奨）
code .
# → コマンドパレット: "Dev Containers: Reopen in Container"

# または Docker Compose
docker compose up -d
docker compose exec app sh
```
