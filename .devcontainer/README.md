# Dev Container

このディレクトリには、Visual Studio Code の Dev Container 設定が含まれています。

## 使い方

プロジェクトのルート README.md を参照してください：
[../README.md](../README.md#dev-containerで開発推奨)

## 構成

- **devcontainer.json** - Dev Container メイン設定
- **compose.yaml** と **compose.override.yaml** を使用

Dev Container は `compose.yaml` の `app` サービスを使用します。
